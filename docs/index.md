# Java

1. Java 中的顶级父类是 [Object](https://docs.oracle.com/en/java/javase/17/docs/api/java.base/java/lang/Object.html)。

## JVM

| 指令              | 作用                       |
|:----------------|:-------------------------|
| invokestatic    | 调用静态方法                   |
| invokespecial   | 调用实例构造方法，私有方法和父类方法       |
| invokevirual    | 调用虚方法                    |
| invokeinterface | 调用接口方法，会在运行时确定一个实现该接口的对象 |
